postgres plugin for `Tutor <https://docs.tutor.edly.io>`__
##########################################################

This is a tutor plugin to enable postgresql as pirmary relational database for edx-platform to replace mysql with fresh setup.


Installation
************

.. code-block:: bash

    pip install git+https://github.com/qasimgulzar/tutor-contrib-postgres-.git

Usage
*****

.. code-block:: bash

    tutor plugins enable postgres
